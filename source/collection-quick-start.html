Data Collection Quick StartÂ¶

Note

Here we are assume that you are sitting at a beamline computer that has the software installed and configured. If you are starting from scratch and you have administative access to your machine, see Deployment Details – or contact that DAMA group.

  1. You should be logged in as a “beamline user” (e.g., xf23id1). Open a terminal.

  2. Activate the collection conda environment.:

    source activate collection

    This command makes the data collection software available by adding it your UNIX $PATH.

  3. Start IPython with a profile.:

    ipython --profile=collection

    A “profile” runs code at startup to define useful variables and functions, including the names of motors and detectors.

You are ready to work. See the bluesky documentaion to learn how to run experiments.
