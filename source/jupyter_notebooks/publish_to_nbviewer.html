  1. wget https://github.com/ipython-contrib/IPython-notebook-extensions/raw/master/custom.example.js

  2. cp custom.example.js ~/.ipython/profile_default/static/custom/example.js

  3. edit custom.js #. Uncomment //    IPython.load_extensions('publishing/gist_it') #. Change it to `` IPython.load_extensions(‘gist’)``

  4. Execute the following code in IPython:

    import IPython IPython.html.nbextensions.install_nbextension(‘https://rawgithub.com/minrk/ipython_extensions/master/nbextensions/gist.js‘)

  5. Start ipython notebook

  6. Look for the curvy arrow to the right of the Cell Toolbar box in the Toolbar

  7. Click it!

  8. It will ask for a github OAuth Token which you can create by following these instructions
